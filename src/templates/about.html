About This Application

This Flask application demonstrates a complete CI/CD pipeline with:

Technology Stack:

  • Backend: Python Flask
  • Containerization: Docker
  • Registry: Azure Container Registry
  • Orchestration: Kubernetes on EKS
  • Infrastructure: Terraform
  • CI/CD: Azure DevOps Pipelines

Architecture:

The application is deployed using a microservices architecture on AWS EKS cluster with proper ingress configuration for external access.
